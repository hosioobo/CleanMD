Why CleanMD

For Markdown that already lives in folders and repos.

CleanMD is for READMEs, docs, specs, and notes that should stay as normal files. Open any folder from your filesystem and work with what is already there.

What it does well

Split source + preview, without leaving the app.

  • Editor and rendered output stay visible together.
  • Code fences, tables, links, math, and YAML are easy to verify.
  • Default light and dark views, plus Paper and Cool light themes.
  • Native Mac workflow with local files and no plugin stack.

Install

Download from GitHub, open eyes wide.

Download latest release

CleanMD ships through GitHub Releases. It is ad-hoc signed but not notarized yet, so macOS may ask you to confirm once on first launch.

Download latest release

First-open steps

  1. Download the latest .zip from GitHub Releases and unzip CleanMD.app.
  2. Move the app to Applications for a normal install location.
  3. If Gatekeeper pauses the first launch, Control-click the app, choose Open, and confirm once. That's it.
