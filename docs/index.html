Download with eyes open

CleanMD is currently released through GitHub and is not notarized yet. Builds are packaged and ad-hoc signed.

First-open steps

  1. Download and unzip CleanMD.app.
  2. Move it to Applications if desired.
  3. If Gatekeeper pauses launch, Control-click, choose Open, and confirm once.

Verify first

  • Inspect release notes
  • Browse repository
  • Download demo MP4
