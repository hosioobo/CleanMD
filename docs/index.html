Install trust

The honest release story, written for a cautious Mac user.

What the release is

CleanMD ships as an ad-hoc signed zip today and is not notarized yet. That means macOS Gatekeeper can warn on first launch even though the full source and release history are public.

How to open it

  1. Download the latest zip and unzip `CleanMD.app`.
  2. Move the app to `Applications` if you want a normal install location.
  3. If Gatekeeper pauses the first launch, Control-click the app, choose `Open`, and confirm once.

Where to verify it

  • Browse the repository
  • Inspect the latest release notes
  • Ask for install help or product feedback
