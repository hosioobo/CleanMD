Install trust

Plainly explained, before you download.

Signing status today

CleanMD releases are ad-hoc signed for packaging convenience and are not notarized yet. The app is safe to inspect from source, but macOS Gatekeeper can still warn on first launch.

First launch steps

  1. Download the latest zip and unzip `CleanMD.app`.
  2. Move the app to `Applications` if you want a normal install location.
  3. If Gatekeeper pauses the first launch, Control-click the app, choose `Open`, and confirm once.
