Who it is for

Built for Markdown that already lives in your folders and repos.

CleanMD is for developers and technical writers who edit READMEs, specs, release notes, docs, and notes as plain files. It is strongest when the job is “open a folder and work” rather than “build a knowledge system.”

Why it wins

Write in source. Review in rendered form.

Keep the text editor and rendered preview visible at the same time so code fences, tables, links, and math are easy to verify before you commit, ship, or share.

  • file explorer and history for fast local navigation
  • syntax highlighting and KaTeX for technical docs
  • offline bundled renderer assets for predictable preview behavior
  • appearance presets when you need a cleaner reading surface

Technical proof

Show technical Markdown, not just prose.

The technical-doc story should be obvious: side-by-side source and rendered preview, real tables and math, and no need to leave the app to check whether the final output is correct.

Why this instead

Lighter than a knowledge base. More inspectable than a drafting canvas.

CleanMD is not trying to replace every notes workflow. It is for people who want a native Mac workspace for local Markdown editing and rendered review.

Install trust

Straightforward GitHub download, with the current caveat explained.

Download with eyes open

CleanMD is currently released through GitHub and is not notarized yet. Builds are packaged and ad-hoc signed, so first launch may require an extra macOS confirmation step.

First-open steps

  1. Download the latest zip and unzip `CleanMD.app`.
  2. Move the app to `Applications` if you want a normal install location.
  3. If Gatekeeper pauses the first launch, Control-click the app, choose `Open`, and confirm once.

Verify before you install

  • Inspect the latest release notes
  • Browse the repository
  • Ask for install help or product feedback

Final CTA

See the app, then decide.

Review the screenshots, confirm the install path, and download the latest build if this is the Markdown workflow you already wish macOS had.
