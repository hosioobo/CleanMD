Install trust

Download from GitHub, with the first-open caveat spelled out.

Download with eyes open

CleanMD is currently released through GitHub and is not notarized yet. Builds are packaged and ad-hoc signed, so first launch may require an extra macOS confirmation step.

First-open steps

  1. Download the latest zip and unzip `CleanMD.app`.
  2. Move the app to `Applications` if you want a normal install location.
  3. If Gatekeeper pauses the first launch, Control-click the app, choose `Open`, and confirm once.

Verify before you install

  • Inspect the latest release notes
  • Browse the repository
  • Ask for install help or product feedback
